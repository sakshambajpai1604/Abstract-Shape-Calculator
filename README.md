# Abstract Shape Calculator
Objective: Create an abstract class Shape with abstract method area(). Implement Circle, Rectangle, and Triangle.

## Create Shape.java
Abstract class with color field, constructor, and abstract area() method

## Implement Subclasses
Create Circle, Rectangle, and Triangle extending Shape

## Write Main Class
Create an ArrayList<Shape>, add one of each, and print their areas
